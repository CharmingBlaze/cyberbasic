**Make a 2D game:**
- Read [Game Development Guide](GAME_DEVELOPMENT_GUIDE.md)

**Make a 3D game:**
- Read [Game Development Guide](GAME_DEVELOPMENT_GUIDE.md)
- Study [3D Graphics API Guide](3D_GRAPHICS_GUIDE.md)
- Check `examples/3d_game_template.bas`

**Use advanced features:**
- Read [Advanced Features Guide](ADVANCED_FEATURES_GUIDE.md)
- Check specific guides (ECS, State System)
- Review examples

**Distribute my game:**
- Read [Distribution Guide](DISTRIBUTION_GUIDE.md)
- Follow platform-specific instructions
- Test on clean systems

**Find examples:**
- Browse `examples/` directory (69+ examples)
- Check `pong_game.bas` for complete game
- See `space_defender.bas` for advanced game
